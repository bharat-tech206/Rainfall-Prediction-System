

### RAINFALL PREDICTION SYSTEM ###

## ABSTRACT

The Rainfall Prediction System is a machine learning project developed using Python and Scikit-learn to predict whether rainfall will occur based on weather conditions. The project analyzes historical weather data, performs data preprocessing and exploratory data analysis, and uses a trained machine learning model to generate predictions.

This project was created to gain practical experience in data analysis, machine learning, model optimization, and predictive analytics.

## Features

* Data cleaning and preprocessing
* Handling missing values
* Exploratory Data Analysis (EDA)
* Data visualization using Matplotlib and Seaborn
* Class imbalance handling through resampling
* Rainfall prediction using Random Forest Classifier
* Hyperparameter tuning using GridSearchCV
* Cross-validation for model evaluation
* Model saving and loading using Pickle
* Prediction on new weather data

## Technologies Used

## Algorithm that applies in this

The project uses the **Random Forest Classifier**, an ensemble machine learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

## Features of Dataset

The model uses weather-related attributes such as:

* Dew Point
* Humidity
* Cloud Cover
* Sunshine
* Wind Direction
* Wind Speed

These features are used to predict whether rainfall is likely to occur.
